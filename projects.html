DriveTest Openings
Aug 2020
DriveTest Openings is a content aggregation website built with the MERN stack. It displays all available dates for booking a road test in an easy to search table.
  • Used NodeJS/Express for the backend, working with a MongoDB database using Mongoose
  • Built the frontend with ReactJS/ReactDOM
    • Implemented client-side filtering and sorting, decreasing server load and time taken for the action
  • Worked with the Pug templating engine to design an alternative lightweight front-end to improve accessibility
  • Wrote a Python script to scrape data from drivetest.ca, and used PyMongo to insert the data into a MongoDB Atlas database

Punchline Guesser
Jul 2020
Panchiraino is a web-based punchline guessing game. Players are given a random joke setup, with the objective to guess the correct punchline.
  • Developed the backend with Python/Django, working with a SQLite database in development, and a PostgreSQL database in production
    • Utilized sessions framework to support anonymous users
    • Implemented account creation/authentication, as well as joke create, update, delete functionality using built-in class-based generic views
  • Designed the frontend with HTML/CSS/Bootstrap/jQuery
  • Used FuzzyWuzzy and nltk to determine the semantic similarity between the expected punchline and the player's guess
  • Loaded Kaggle joke dataset into the database using Python scripts
  • Customized admin page for easy site maintenance
